ECONOMIZE AJUDANDO O MEIO AMBIENTE

Projeto

Nosso projeto tem como objetivo monitorar o nivel das lixeiras, impedindo que fique super lotada, assim podendo fazer com que o cliente tomas as devidas decisões.

Como Funciona

Nos instalaremos um sensor arduino dentro de cada lixeira, assim tendo em tempo real, o nível de cada uma..

Benefícios

O cliente tera uma dashboard disponivel a qualquer momento, com atualizações a tempo real dos niveis de lixo de cada lixera e de cada rota.
